# PotionStore

Installation script containing **tools** for web application penetration testing.  

## Usage: `potions.sh`

1. Download 

~~~bash
git clone https://github.com/mattmasel/potionstore.git
~~~

2. Move to potionstore

~~~bash
cd potionstore
~~~

3. Allow execution

~~~bash
chmod +x potions.sh
~~~

4. Run the script to install tools!

~~~bash
./potions.sh
~~~
